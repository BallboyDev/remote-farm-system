# Smart Farm IoT Platform

> Mobile Web + Personal Server + LoRa 기반 스마트 농업 제어 시스템

## 개요

본 프로젝트는 클라우드 서비스에 의존하지 않고, 개인이 구축한 서버를 통해 농업 장비를 원격 제어하고 상태를 모니터링하는 스마트팜 플랫폼이다.

모바일 웹 또는 앱을 통해 HTTPS로 농장 관리 서버에 접근하며, 서버는 Raspberry Pi에 연결된 LoRa HAT을 이용하여 현장의 Heltec WiFi LoRa 32 장치들과 통신한다.

각 Heltec 노드는 독립적인 제어 장치(관수, 환풍기, 센서 등)로 동작하며, 릴레이와 센서를 통해 실제 농업 장비를 제어한다.
